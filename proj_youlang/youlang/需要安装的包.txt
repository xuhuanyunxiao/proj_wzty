关于Python安装官方whl包和tar.gz包的方法详解(收藏)

Windows环境：

　　安装whl包：pip install wheel -> pip install **.whl

　　安装tar.gz包：cd到解压后路径,python setup.py install

Linux环境：

　　安装whl同上

　　安装tar.gz:cd到解压后路径,./configure -> make -> make install
